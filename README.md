## 🚀 Featured Projects

<details>
<summary><b>🤖 DevTeam.AI — Autonomous Software Development System</b></summary>
<br/>

> *Give it a product idea. Get back a production-ready codebase.*

A multi-agent system where specialized AI agents **collaborate like a real dev team** — a PM agent breaks down requirements, architect agents design the system, coder agents implement features in parallel, a QA agent writes and runs tests, and a DevOps agent handles deployment configs.

**The core insight:** most "AI coding tools" still require a human in the loop at every step. DevTeam.AI eliminates that.

- 🧠 Orchestrated with **LangGraph** (stateful, cyclical agent graphs)
- 🔁 Inter-agent communication via shared memory + message passing
- 🛠️ Tool-use: file I/O, code execution sandbox, GitHub API
- 📡 Real-time progress streaming via **WebSockets**
- 🗃️ Persistent state across sessions with **PostgreSQL**

**Stack:** `LangGraph` `Claude 3.5 Sonnet` `GPT-4o` `FastAPI` `Next.js` `Docker` `PostgreSQL` `Redis`

</details>

> *Not a chatbot. An agent that resolves, escalates, and learns.*
